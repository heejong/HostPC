How to start OpenPET Host PC development

1) Host PC development is done primarily in ANSI C through LabWindows. 
2) Documentation is created by including Doxygen-style comments.
3) Collaboration is accomplished through sharing software on Github.

We have included documents to help you get started with each of these services in this folder.
We recommend reading this documentation in the order listed (LabWindows -> Doxygen -> Github).
